## Gokapi

A Go API and Vite/Bun/React Web App.

- JWT based auth over HTTPS
- Go Server Refreshing with AIR
- HMR with Vite on port:3000 (HTTP proxy to 8443)
- production on port:8443 (HTTPS)
- realtime websocket state
